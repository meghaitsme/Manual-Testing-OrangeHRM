 Overview
This project showcases manual testing of the [OrangeHRM](https://www.orangehrm.com/) system, focusing on structured testing methodologies, defect reporting, and test documentation. The repository contains test plans, test cases, and bug reports to demonstrate proficiency in manual testing.

<img width="708" height="385" alt="overall status" src="https://github.com/user-attachments/assets/5c44babd-62b8-406b-82a2-62b58e6073de" />


 Repository Contents
- Test Plans – Defines the testing strategy, scope, objectives, and approach.
- Test Cases – A detailed set of test cases covering various functionalities of the OrangeHRM Admin Module.
- Bug Reports – A documented list of identified defects, including severity, priority, and reproduction steps.

 Testing Scope
- Application Tested: OrangeHRM (Admin Module)
- Testing Types:  
  - Functional Testing  
  - UI Testing  
  - Regression Testing
